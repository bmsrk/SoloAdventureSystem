## ?? Navigation Tips

### Always Start at the Hub
**[docs/INDEX.md](./INDEX.md)** is your main entry point.

### Each Category Has an Index
Every folder has a `README.md` that:
- Lists all documents in that category
- Explains what each doc covers
- Provides quick navigation
- Links to related categories

### Follow the Breadcrumbs
Every document ends with:
```markdown
---
*Part of the [Solo Adventure System Documentation](../INDEX.md)*
```

Click to return to the main index.

### Use "See Also" Sections
Documents link to related topics at the bottom.

## ?? File Naming Convention

- **Directories**: `lowercase-with-dashes/`
- **Documents**: `UPPER_SNAKE_CASE.md`
- **Index files**: `README.md`

**Examples**:
- ? `getting-started/QUICK_START.md`
- ? `user-guides/WORLD_CUSTOMIZATION.md`
- ? `technical/API_REFERENCE.md`
- ? `getting_started/quick_start.md` (wrong casing)
